# Shortlist ambassadeurs Adventurer — V1

Objectif : 20 ambassadeurs actifs d'ici 90 jours pour lancer la communauté
dans des niches clairement identifiées (sport + géo).

Format d'approche : DM Instagram personnalisé + email de suivi à J+2.
Budget : 5 € / inscription réussie + 50 € de matos Adventurer (sticker pack, gourde, t-shirt) + accès early access.

## Profils-types à viser

| Segment | Sport | Zone | Pourquoi ce profil |
|---|---|---|---|
| Micro-influenceur (5k-30k) | Kitesurf | Tarifa, Leucate, Dakhla | Tribu locale active, turnover de riders |
| Coach / moniteur certifié | Apnée (AIDA), plongée (PADI) | Nice, Marseille, Bali | Crédibilité sécurité + accès clients premium |
| Athlète semi-pro | Trail / ultra | Chamonix, Annecy, Courmayeur | Saison UTMB, énorme engagement |
| Créateur YouTube | Parapente, ski de rando | Alpes du Nord | Audience très ciblée |
| Guide de haute montagne (UIAGM) | Alpinisme | Zermatt, Chamonix | Monétisation directe via bookings |
| Rider gravel / bikepacking | Vélo | Pyrénées, Cévennes | Communauté montante, peu d'apps dédiées |
| Club / association | Escalade | Fontainebleau, Ardèche | Active toute l'année, multi-niveaux |

## 20 comptes Instagram à contacter en priorité (cibles publiques, ordre indicatif)

> ⚠️ À valider avant contact : vérifier que le compte est actif en 2026,
> que l'audience est cohérente, qu'il n'est pas déjà sponsorisé par Strava/Komoot exclusivement.

### Mer (8 cibles)
1. Riders kitesurf Tarifa (comptes type @tarifa_kitesession, écoles locales)
2. Riders Leucate / Narbonne-Plage
3. Riders Dakhla / Essaouira (communauté francophone)
4. Apnée Bleue / Blue Vision / AIDA France (instructeurs Nice/Marseille)
5. Wing foil Bretagne / La Torche
6. Surf Hossegor / Capbreton
7. Voile mini-transat / Figaro (profils aventuriers pas top 10)
8. Paddle Corse / Côte d'Azur

### Terre (8 cibles)
9. Traileurs UTMB (finishers qui postent, pas top 20)
10. Ultra-trail Cévennes / Ardèche
11. Alpinistes Chamonix / guides UIAGM
12. Escalade Fontainebleau bloc
13. Gravel / bikepacking Pyrénées
14. Ski de rando Aravis / Beaufortain
15. Randonnée slow / fastpacking (GR20, HRP)
16. VTT enduro Finale Ligure / Alpes du Sud

### Air (4 cibles)
17. Parapente Saint-Hilaire / Annecy
18. Speedriding / speedflying Chamonix
19. ULM / vol à voile Pyrénées
20. Base jump / wingsuit (seulement profils sécurité-responsables, éviter thrill-seekers)
